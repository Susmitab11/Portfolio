Project Overview

The Cafe Billing System is a JavaScript-based web application that allows users to place food and beverage orders and generate a bill dynamically. Users can view the available menu, enter an item name and quantity, add items to the bill, calculate the total amount, and clear the bill when needed. The project was built to practice JavaScript fundamentals, DOM manipulation, event handling, and creating an interactive user interface.

Key Features

  • Interactive café menu display
  • Add menu items to the bill
  • Quantity-based price calculation
  • Dynamic bill generation
  • Automatic total bill calculation
  • Clear bill functionality
  • User-friendly interface

Challenges & Solutions

Challenge

Keeping track of multiple ordered items while updating the bill correctly was one of the main challenges.

The total amount needed to update automatically whenever a new item was added.

The application needed an easy way to start a new order without refreshing the page.

Solution

Used JavaScript to capture user input, process each order, and dynamically update the bill section using DOM manipulation whenever the Add to Bill button was clicked.

Implemented JavaScript logic to calculate the running total and display the updated amount in the Total section after every order.

Added a Clear Bill button that removes all bill items and resets the total amount using JavaScript.

  • Manipulating the DOM to update webpage content dynamically
