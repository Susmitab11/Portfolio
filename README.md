Personal Portfolio
A responsive personal portfolio website showcasing my projects, technical skills, certifications, and journey as a Frontend Developer. 
The portfolio serves as a central place where recruiters and developers can explore my work, view live projects, and learn more about my experience.

About the Project:
This portfolio was designed and developed to present my frontend development skills through a clean, responsive, and user-friendly interface. It highlights 
my featured projects, technical expertise, certifications, and contact information while providing an easy navigation experience across different devices.

Features:
- Responsive design for desktop, tablet, and mobile devices
- Modern and clean user interface
- Featured projects with detailed project pages
- Live Demo and GitHub links for each project
- Technical skills section
- Education and certifications
- Contact section with social links
- Smooth navigation between sections

Technology Used:
HTML
CSS
Responsive Design
