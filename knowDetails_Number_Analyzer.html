Project Overview

The Number Analysis Tool is a JavaScript-based web application that analyzes numbers within a user-defined range. Users enter a start number and an end number, and the program checks each number for divisibility by 3, 5, and 7. Based on the divisibility, it displays labels such as Fizz, Buzz, Seven, or their combinations. The application also provides a summary showing how many numbers are divisible by 3, 5, and 7 within the selected range.

Key Features

  • User-defined number range analysis
  • Divisibility checks for 3, 5, and 7
  • Displays Fizz, Buzz, Seven, and combined outputs
  • Summary count of divisible numbers
  • Dynamic result display using DOM manipulation

Challenges & Solutions

Challenge

Some numbers are divisible by multiple values (3, 5, and 7). If individual conditions were checked first, the program would return incorrect results and skip more specific combinations.

Displaying the generated results and summary on the webpage while keeping the interface interactive was another challenge.

Solution

The conditions were arranged from the most specific to the least specific: Divisible by 3, 5, and 7 Divisible by 3 & 5 Divisible by 3 & 7 Divisible by 5 & 7 Divisible by 3 Divisible by 5 Divisible by 7

This ensured accurate and efficient output for every number.

Used DOM manipulation with document.getElementById() to access HTML elements, attached an onclick event to the Analyze button, processed the numbers using a while loop, and updated the webpage dynamically with the analysis results and summary.
